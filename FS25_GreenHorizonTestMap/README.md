# Green Horizon Test Map

This is a developer-only FS25 map used to test outdoor HEMP alongside Green Horizon Industries.

Install both ZIPs:

1. `FS25_GreenHorizonIndustries.zip`
2. `FS25_GreenHorizonTestMap.zip`

Start a **new save**, select **Green Horizon Test Map**, and enable both mods. The map reuses the official Riverbend Springs terrain, fields, shop, and starting equipment while adding a map-owned HEMP foliage registration.

First test:

1. Check that HEMP appears in the crop calendar and Great Plains seeder selection.
2. Sow HEMP and inspect every growth state.
3. Harvest mature HEMP with the MF 8570 and compatible grain header.
4. Confirm HEMP enters the combine tank and the field changes to the harvested state.
5. Save, reload, and check `log.txt`.

This first build intentionally has no HEMP windrow. Harvest output is bulk `HEMP` for delivery to the existing CBD production chain.
