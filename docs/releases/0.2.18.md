# Green Horizon Industries 0.2.18.0

## Active Test Feature

The Hemp Greenhouse remains the only active placeable. Field hemp, crop icons, cutter effects, custom pallets, and processing recipes remain excluded from active gameplay XML.

## Added

- Transparent source-icon generator for six hemp fill types plus crop and calendar icons.
- Inactive icon manifest with activation requirements.
- Cutter-effect source generator for chaff, stem shards, leaf fragments, and dust.
- Inactive cutter-effect plan with mature-state routing, effect profiles, particle budgets, and sound targets.
- Inactive map-registration draft synchronizing field state channels and harvest transitions.
- Dedicated PowerShell validator for fruit, foliage, growth, icons, cutter routing, and map integration.
- One-command Blender generation pipeline for greenhouse, field, cutter, icon, and pallet source assets.
- Automatic Blender installation detection with per-generator logs and expected-output checks.
- Strict greenhouse i3d export validator.
- Project-status command that reports the next required action.

## Updated

- Foliage plan to version 3 with icon, cutter-effect, and map-registration references.
- Main preflight checker to validate the new files and call the field-foundation validator.
- Packaging requirements to include all inactive Phase 2.18 XML manifests and plans.
- Packaging now refuses to continue when the placeholder or an incomplete greenhouse export is present.
- Windows test menu now exposes generation, export validation, documentation, and status options.
- GIANTS Editor instructions now match the one-script peaked-roof model and deterministic helper hierarchy.
- Documentation and project version remain `0.2.18.0`.

## Deliberately Not Linked

- No icon paths were added to `fillTypes.xml` or `fruitTypes.xml`.
- `fruitTypes.xml` remains excluded from `modDesc.xml`.
- Cutter-effect textures are not connected to any vehicle or cutter XML.
- No map foliage or density-map registration was activated.
- Generated pallet XML remains unlinked from active fill types.

## Recommended Hands-On Flow

1. Run menu option `15` to see the current next action.
2. Run option `12` to generate all source assets.
3. Inspect and manually export the greenhouse root.
4. Run option `13` to validate the i3d and shapes files.
5. Run option `1` for full preflight.
6. Run option `3` to package and install.
7. Test in FS25 and run option `4` for the filtered log.
